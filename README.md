При нажатии /start выводится 2 кнопки, зарегистрироваться и валюта.
При нажатии зарегистрироваться бот выносит в базу данных user,id,first name, last name пользователя.
При нажатии валюта бот выводит курс валюты в Российских рублях(доллар, евро, юань, тенге).
После регистрации пользователь может изменить свои данные, а админ посмотреть данные всех пользователей.
